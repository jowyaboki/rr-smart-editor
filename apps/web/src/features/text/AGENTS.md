# Text & Typography Engine Implementation Details

The text engine manages high-fidelity typography objects that are rendered as first-class citizens in the video editor.

## Core Models

- **TextObject**: The main entity containing content, style, layout, and transform data.
- **TextStyle**: Defines typographic properties (font, weight, stroke, shadow).
- **TextLayout**: Defines how text flows within its container (wrapping, alignment).

## Key Services

- **TextEngine**: Converts a `TextObject` into a standard React/CSS style object for rendering.
- **TypographyService**: Provides factory methods for creating standardized text objects.
- **TextLayoutService**: (Foundations) for calculating text bounds and overflow.

## Remotion Integration

The `TextLayer` component in the Remotion tree consumes the `TextObject` and applies the styles generated by the `TextEngine`. This ensures the preview and the final render are identical.

## UI Integration

The `TypographyPanel` in the Inspector provides a professional suite of controls for real-time editing of all typographic and layout properties.
